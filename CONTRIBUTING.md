# Contributing

1. **Skill behavior** → edit `SKILL.md` + `references/`.
2. **Visual tokens** → `references/ui-tokens.md` (keep Lab·Canonical as default).
3. **Docs / launch** → `README.md` / `README.zh-CN.md` / `docs/LAUNCH-CHECKLIST.md`.

Please keep generated `skill-lab/*.html` out of PRs unless they are **sanitized examples** (no private URLs); `.gitignore` ignores local outputs by default.
