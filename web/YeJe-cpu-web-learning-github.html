README angles

  • Demand: Skill repos are easy to star, hard to “see the wiring” without a single scrollable map.
  • Spread: One HTML file; works offline after fonts cache; fits GitHub Pages under /web if you want.
  • Caveat: GitHub counters are a snapshot at generation time.
  • Compliance: Do not embed secrets or private URLs in generated pages.
  • 需求：Skill 仓好收藏，调用链若没有单页地图则不够直观。
  • 传播：单文件；字体缓存后可离线；也可配合 GitHub Pages 指到 /web。
  • 注意：GitHub 数字为生成时刻快照。
  • 合规：生成页勿写密钥或内网地址。
